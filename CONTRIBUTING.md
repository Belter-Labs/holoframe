# Contributing to Holoframe

Thanks for your interest in Holoframe!

## Feature Requests & Bug Reports

We welcome feature requests and bug reports!

**To submit:**
1. Check [existing issues](https://github.com/Belter-Labs/holoframe/issues) first
2. Create a new issue with:
   - Clear description
   - Use case or problem it solves
   - Any relevant examples
